## ğŸ¯ Quick Reference

### Common Tasks

```bash
# Add a new screen
1. Create function component in App.tsx
2. Add to detailScreen conditional rendering
3. Add navigation trigger from parent screen
4. Implement back navigation

# Add new action category
1. Add to categories array in TrackScreen
2. Add actions with point values
3. Test handleAddAction flow

# Add new reward
1. Add to rewards array in RewardsScreen
2. Set points cost and category
3. Test redemption flow

# Modify styling
1. Edit Tailwind classes in components
2. Check globals.css for design tokens
3. Maintain wireframe aesthetic (black/white)

# Add animation
1. Import motion from 'motion/react'
2. Wrap element in <motion.div>
3. Add initial, animate, exit props
4. Use AnimatePresence for conditional rendering
```
